# Grok 视频生成

Grok 视频模型使用 `/v1/videos` 协议，支持文生视频、首帧生视频和参考图生视频。它虽然兼容 OpenAI 风格的鉴权与请求方式，但在菜单中按独立模型归类。

## 接口地址

```text
POST https://fullstacklab.xyz/v1/videos
GET  https://fullstacklab.xyz/v1/videos/{task_id}
```

## 模型列表

| 模型 | 能力 |
| --- | --- |
| grok-video-1.0 | 文生视频、首帧生视频、参考图生视频 |
| grok-video-1.5 | 仅首帧生视频 |

## 请求参数

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| model | string | 模型名，例如 `grok-video-1.0` |
| prompt | string | 视频提示词 |
| duration | integer | 视频时长 |
| resolution | string | `480p` 或 `720p` |
| aspect_ratio | string | `16:9`、`9:16`、`1:1`、`3:2`、`2:3` |
| preset | string | 通常为 `normal` |
| images | string[] | 可选参考图 URL 或 Base64，首帧或参考图生视频时使用 |

## Shell 示例

```bash
curl "https://fullstacklab.xyz/v1/videos" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "grok-video-1.0",
    "prompt": "小猫坐在河边钓鱼，镜头缓慢推进",
    "duration": 6,
    "resolution": "720p",
    "aspect_ratio": "16:9",
    "preset": "normal"
  }'
```

## Python 示例

```python
import requests

response = requests.post(
    "https://fullstacklab.xyz/v1/videos",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "grok-video-1.0",
        "prompt": "小猫坐在河边钓鱼，镜头缓慢推进",
        "duration": 6,
        "resolution": "720p",
        "aspect_ratio": "16:9",
        "preset": "normal",
    },
    timeout=300,
)
response.raise_for_status()
print(response.json())
```

## Java 示例

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class GrokVideoExample {
    public static void main(String[] args) throws Exception {
        String body = """
            {
              "model": "grok-video-1.0",
              "prompt": "小猫坐在河边钓鱼，镜头缓慢推进",
              "duration": 6,
              "resolution": "720p",
              "aspect_ratio": "16:9",
              "preset": "normal"
            }
            """;
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://fullstacklab.xyz/v1/videos"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(body))
            .build();
        System.out.println(HttpClient.newHttpClient()
            .send(request, HttpResponse.BodyHandlers.ofString()).body());
    }
}
```

## 结果轮询

创建成功后使用 `GET /v1/videos/{task_id}` 轮询。完成后兼容读取 `url`、`video_url`、`result_url` 或 `metadata.result_url`。
