# API Overview

Fullstack Lab provides OpenAI-compatible, Anthropic-compatible, Gemini-compatible, image generation, and video generation endpoints.

## Base URLs

General base URL:

```text
https://fullstacklab.xyz
```

OpenAI-compatible base URL:

```text
https://fullstacklab.xyz/v1
```

Use `YOUR_API_KEY` in examples. Do not publish real keys.
