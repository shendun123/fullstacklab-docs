# Before Purchase

Before using Fullstack Lab, confirm your scenario:

- You only need Claude Code.
- You only need Codex.
- You need both Claude Code and Codex.
- You want to call the API directly from your own application.
- You want to use a desktop client such as Cherry Studio.

For first-time users, start with a small balance or a basic quota. Make sure your API Key, Base URL, and target tool all work correctly before increasing usage.

Fullstack Lab balance is platform-side usage credit. It is not the same as a Claude official account balance or an OpenAI official account balance.

## Base URLs

General platform base URL:

```text
https://fullstacklab.xyz
```

OpenAI-compatible base URL:

```text
https://fullstacklab.xyz/v1
```

## Security Reminder

Never share your full API Key in screenshots, public chats, GitHub repositories, or documentation.
