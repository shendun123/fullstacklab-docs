# Knowledge Cutoff Test

Do not judge model quality only by asking:

```text
What is your knowledge cutoff?
What model are you?
```

These answers may be affected by system prompts, client settings, or model self-reporting limitations.

## Better Evaluation Methods

Use real tasks:

- Fix a real bug
- Generate working code
- Follow complex instructions
- Handle long context
- Compare outputs across multiple runs
- Verify API behavior with real requests

A model's self-reported knowledge date is not a reliable benchmark by itself.
