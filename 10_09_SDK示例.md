# SDK 示例

## Shell

```bash
curl "https://fullstacklab.xyz/v1/chat/completions" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-5.5",
    "messages": [
      {"role": "user", "content": "Hello"}
    ]
  }'
```

## Python

```python
import requests

api_key = "YOUR_API_KEY"

res = requests.post(
    "https://fullstacklab.xyz/v1/chat/completions",
    headers={
        "Authorization": f"Bearer {api_key}",
        "Content-Type": "application/json",
    },
    json={
        "model": "gpt-5.5",
        "messages": [{"role": "user", "content": "Hello"}],
    },
    timeout=60,
)

res.raise_for_status()
print(res.json()["choices"][0]["message"]["content"])
```

## Java

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class RestExample {
    public static void main(String[] args) throws Exception {
        String apiKey = "YOUR_API_KEY";
        String body = """
            {
              "model": "gpt-5.5",
              "messages": [
                {"role": "user", "content": "Hello"}
              ]
            }
            """;

        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://fullstacklab.xyz/v1/chat/completions"))
            .header("Authorization", "Bearer " + apiKey)
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(body))
            .build();

        HttpResponse<String> response = HttpClient.newHttpClient()
            .send(request, HttpResponse.BodyHandlers.ofString());

        System.out.println(response.body());
    }
}
```

## 列出模型

Shell：

```bash
curl "https://fullstacklab.xyz/v1/models" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Python：

```python
import requests

res = requests.get(
    "https://fullstacklab.xyz/v1/models",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    timeout=30,
)
res.raise_for_status()
print(res.json())
```

Java：

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class ModelsExample {
    public static void main(String[] args) throws Exception {
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://fullstacklab.xyz/v1/models"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .GET()
            .build();

        HttpResponse<String> response = HttpClient.newHttpClient()
            .send(request, HttpResponse.BodyHandlers.ofString());

        System.out.println(response.body());
    }
}
```

## REST 请求建议

对于图像、视频、任务轮询类接口，优先使用 REST 调用，方便打印完整状态码、返回内容、任务 ID 和错误详情。示例语言统一保留 Shell、Python、Java 三种。

## 代码安全建议

示例代码中不要写真实 Key。推荐使用环境变量：

```python
import os

API_KEY = os.getenv("FULLSTACKLAB_API_KEY")
```

Shell 设置环境变量示例：

```bash
export FULLSTACKLAB_API_KEY="你的真实Key"
```
