# SDK 示例

## OpenAI Python SDK

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://fullstacklab.xyz/v1"
)

response = client.chat.completions.create(
    model="gpt-5.5",
    messages=[{"role": "user", "content": "Hello"}]
)

print(response.choices[0].message.content)
```

## Anthropic Python SDK

```python
from anthropic import Anthropic

client = Anthropic(
    api_key="YOUR_API_KEY",
    base_url="https://fullstacklab.xyz/v1"
)

response = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello"}]
)

print(response.content[0].text)
```

## REST 请求建议

对于图像、视频、任务轮询类接口，优先使用 REST 调用，方便打印完整状态码、返回内容、任务 ID 和错误详情。

## 代码安全建议

示例代码中不要写真实 Key。推荐使用环境变量：

```python
import os

API_KEY = os.getenv("FULLSTACKLAB_API_KEY")
```

Windows 设置用户环境变量示例：

```powershell
[Environment]::SetEnvironmentVariable("FULLSTACKLAB_API_KEY", "你的真实Key", "User")
```
