# 接口错误与排查

本页汇总文本、图像和视频接口常见问题。

## 通用状态码

| 状态码 | 说明 | 处理建议 |
| --- | --- | --- |
| 400 | 请求参数错误 | 检查字段名、模型名、JSON 格式和必填参数 |
| 401 | 认证失败 | 检查 API Key 是否正确，是否复制完整 |
| 403 | 权限不足 | 检查账号分组、模型权限、服务是否开通 |
| 429 | 请求频率超限 | 降低并发，稍后重试 |
| 500 | 服务端错误 | 等待后重试，或联系平台排查 |
| 502 / 503 / 504 | 上游或网关异常 | 等待、重试、切换模型或联系平台 |
| 522 / 524 | 连接或超时问题 | 检查网络、Cloudflare、源站超时设置 |

## 图片接口排查

| 现象 | 可能原因 | 处理方式 |
| --- | --- | --- |
| 连接失败 | 地址、端口、网络、防火墙异常 | 确认 Base URL 和网络连通性 |
| 请求超时 | 生图耗时较长或上游慢 | 提高 timeout，例如 60 / 120 / 300 秒 |
| 返回没有图片 | 返回字段不同 | 同时兼容 `url` 和 `b64_json` |
| 图生图失败 | 参考图不可访问 | 使用公网可访问的图片直链 |

## 任务式接口排查

| 现象 | 可能原因 | 处理方式 |
| --- | --- | --- |
| 返回没有 task_id/id | 创建任务失败或返回格式变化 | 打印完整返回内容 |
| 一直 queued / processing | 排队或上游处理慢 | 拉长轮询超时时间 |
| failed | 提示词、模型、权限或上游失败 | 查看 error、code、fail_reason |
| completed 但没有地址 | 返回字段不同 | 兼容 `url`、`output`、`video_url` |

## 安全提醒

- GitHub 仓库
- 文档站
- 截图
- 公开群聊
- 给用户的示例代码

