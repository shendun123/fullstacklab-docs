# Vidu 视频生成

Vidu 支持文生视频、首帧生视频和多参考图生视频，统一通过异步任务接口调用。

## 接口地址

```text
POST https://fullstacklab.xyz/v1/videos
GET  https://fullstacklab.xyz/v1/videos/{task_id}
```

## 支持模型

### 文生视频

```text
vidu/viduq3-pro_text2video
vidu/viduq3-turbo_text2video
vidu/viduq2_text2video
```

### 首帧生视频

```text
vidu/viduq3-pro_img2video
vidu/viduq3-turbo_img2video
vidu/viduq2-pro-fast_img2video
vidu/viduq2-pro_img2video
vidu/viduq2-turbo_img2video
```

### 参考图生视频

```text
vidu/viduq3-mix_reference2video
vidu/viduq3_reference2video
vidu/viduq3-turbo_reference2video
vidu/viduq2-pro_reference2video
vidu/viduq2_reference2video
```

## 文生视频参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| model | string | 是 | 文生视频模型名 |
| prompt | string | 是 | 视频提示词 |
| duration | integer | 是 | 视频时长 |
| resolution | string | 是 | `540P`、`720P`、`1080P` |
| aspect_ratio | string | 是 | `16:9`、`9:16`、`1:1`、`4:3`、`3:4` |
| preset | string | 是 | 通常使用 `normal` |

## 首帧与参考图参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| model | string | 是 | img2video 或 reference2video 模型 |
| prompt | string | 是 | 视频提示词 |
| duration | integer | 是 | 通常为 1 至 16 秒 |
| size | string | 否 | 如 `1280x720` |
| metadata.resolution | string | 是 | `540P`、`720P`、`1080P` |
| metadata.watermark | boolean | 否 | 是否添加“内容由 AI 生成”水印 |
| metadata.audio | boolean | 否 | 是否生成音频，仅部分模型支持 |
| images | string[] | 是 | 公网参考图片 URL |

## Shell 示例

```bash
curl "https://fullstacklab.xyz/v1/videos" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "vidu/viduq3_reference2video",
    "prompt": "镜头围绕主体缓慢环绕",
    "duration": 6,
    "size": "1280x720",
    "metadata": {
      "resolution": "720P",
      "watermark": false,
      "audio": true
    },
    "images": ["https://example.com/reference.jpg"]
  }'
```

## Python 示例

```python
import requests

response = requests.post(
    "https://fullstacklab.xyz/v1/videos",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "vidu/viduq3-pro_text2video",
        "prompt": "小猫坐在河边钓鱼，镜头缓慢推进",
        "duration": 6,
        "resolution": "720P",
        "aspect_ratio": "16:9",
        "preset": "normal",
    },
    timeout=300,
)
response.raise_for_status()
print(response.json())
```

## Java 示例

```java
import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;

public class ViduExample {
    public static void main(String[] args) throws Exception {
        String body = """
            {
              "model": "vidu/viduq3-pro_text2video",
              "prompt": "小猫坐在河边钓鱼，镜头缓慢推进",
              "duration": 6,
              "resolution": "720P",
              "aspect_ratio": "16:9",
              "preset": "normal"
            }
            """;
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://fullstacklab.xyz/v1/videos"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(body))
            .build();
        System.out.println(HttpClient.newHttpClient()
            .send(request, HttpResponse.BodyHandlers.ofString()).body());
    }
}
```

## 查询结果

创建成功后读取 `id` 或 `task_id`，通过 `GET /v1/videos/{task_id}` 查询。完成后兼容读取 `url`、`video_url` 或 `result_url`。

## 注意事项

- 图片 URL 必须是公网可访问的文件直链。
- `audio` 仅部分 Vidu Q3 模型支持。
- 分辨率字段大小写应与模型要求一致。
